# Huella Cero

Sistema web orientado a la reducción de la huella de carbono mediante el monitoreo de hábitos sostenibles relacionados con transporte y consumo energético.

## Tecnologías utilizadas

- React
- Node.js
- Express
- CSS
- GitHub Actions
- Docker

## Arquitectura del sistema

El sistema se encuentra dividido en:

- Frontend desarrollado en React
- Backend desarrollado con Node.js y Express
- Comunicación mediante API REST
- Base de datos MongoDB

## Instalación local

### Backend

```bash
cd backend
npm install
node server.js
```
